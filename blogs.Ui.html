Bootstrap ဆိုစွာဇာချင့်လဲ?

Bootstrap ဆိုစွာ Build first, responsive sites ဆိုရေ အတိုင်း သုံးရစွာ လွယ်ကူမြန်ဆန်ပနာ responsive အတွက်လည်း အလွယ် တကူစဉ်းစားနိုင်ရေ အတွက် နန့် web Developer တိ လည်း အခု အချိန်ထိ သုံးနီသိရေ framework တစ်ခုလည်းဖြစ်ပါရေ။ 'responsive' လို့ပြောခရေ အတွက်နန့် responsive ဆိုစွာဇာလေ ဆိုစွာပြောကြည့်ကတ်ဖို့ပါ။
responsive ဆိုစွာ Device တိုင်းနန့် လိုက်လျောညီထွေ ဖြစ်တယ်လို့ အလွယ်မှတ်ယူလို့ရပါရေ။ကိုယ်ဖန်တီးလိုက်တယ် website စွာ device တိုင်းမှာ သုံးရစွာ ကြည့်ရှုရစွာ အဆင်ပြေရေဆိုစွာ responsive ဖြစ်တယ် ဆိုပနာ မှတ်နိုင်ပါရေ။ ယောပိုင်ဆိုကေ ' Framework ' ဆိုစွာဇာလေ ဆိုစွာကို ပြောကြည့်ကတ်ဖို့ပါ။
သူများလုပ်ထားရေ Frame အထဲမှာ ဝင်‌သုံးစွာကို frameworkကို သုံးရေ လို့ မှတ်လို့ရပါရေ။ သူများရေးထားရေ Frame လို့ ပြောလိုက်တယ် အတွက်နန့် ‌ယော frame ကိုသုံးချင်ကေ ဖန်တီးထားရေ လူ ၏ rules and regulations တိကိုလိုက်နာဖို့လိုအပ်ပါရေ။ တနည်းအားဖြင့်ဆိုကေ ကိုယ်တိုင် အစကနီဖန်တီးစရာမလိုဘဲနန့် rules and regulations တိကို လိုက်နာပနာ ဖန်းတီးစွာကို framework ကိုသုံးရေလို့ မှတ်ယူလို့ရပါရေ။အခုလက်ရှိ Bootstrap version က 5.3 ဖြစ်ပါရေ။
